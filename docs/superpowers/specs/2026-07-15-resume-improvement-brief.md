## HTML Improvements

-   Use semantic HTML (
    ```{=html}
    <header>
    ```
    ,
    ```{=html}
    <main>
    ```
    ,
    ```{=html}
    <section>
    ```
    ,
    ```{=html}
    <article>
    ```
    ).
-   Keep one H1 only.
-   H2 for sections.
-   H3 for employers.
-   Use UL/LI for skills and achievements.
-   Avoid long paragraphs.
-   Add print-friendly CSS.
-   Ensure mobile responsiveness.
-   Preserve ATS-readable text (avoid rendering important text as
    images).

## Visual Improvements

-   Increase whitespace.
-   Consistent typography.
-   Highlight job titles more than company names.
-   Use subtle separators.
-   Keep color palette minimal.
-   Improve hierarchy with section spacing.
